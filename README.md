## Global Styles Setup (Do This FIRST!)

### Step 1: Set Up Global Colors
Navigate to: **Site Settings → Global Colors**

Add these exact colors:

| Name | Hex Code | Usage |
|------|----------|-------|
| SCS Blue (Primary) | #124CA1 | Headings, buttons, brand elements |
| Ocean Blue | #5795CE | Secondary CTAs, accents |
| Seal Grey | #919294 | Meta text, borders |
| Dark Grey | #555555 | Footer background |
| Light Grey | #D9D9D9 | Borders, dividers |
| Headings Black | #1A1A1A | Main headings |
| Body Text Black | #222222 | Body text |
| Background Grey | #F2F2F2 | Section backgrounds |

### Step 2: Set Up Global Typography
Navigate to: **Site Settings → Global Fonts**

Set ALL fonts to **Poppins** family with these settings:

| Element | Weight | Size | Use Global Color |
|---------|--------|------|------------------|
| H1 - Page Title | 600 (Semi-bold) | 53px | SCS Blue or White |
| H2 - Section Heading | 600 (Semi-bold) | 37px | SCS Blue or White |
| H3 - Subsection | 500 (Medium) | 19px | Headings Black |
| H4 - Minor (Opt A) | 400 (Regular) | 21px | SCS Blue |
| H4 - Minor (Opt B) | 600 (Semi-bold) | 16px | SCS Blue |
| H5 - Locations | 600 (Semi-bold) | 24-27px | SCS Blue |
| H5 - Blogs | 700 (Bold) | 24-27px | SCS Blue |
| H6 - Blog Title | 400 (Regular) | 32px | White |
| Body 1 (Primary) | 500 (Medium) | 16px | SCS Blue or White |
| Body 2 (Standard) | 400 (Regular) | 19px | Headings Black |
| Body 2 (Bolded) | 600 (Semi-bold) | 19px | Headings Black |
| Button Text | 500 (Medium) | 16px | White |
| Caption / Meta | 500 (Medium) | 19-21px | White or Seal Grey |
